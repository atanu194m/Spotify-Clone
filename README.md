# 🎵 Spotify Clone

A hand-crafted, fully functional music player built with pure **HTML**, **CSS**, and **JavaScript** — no frameworks, just music.

![Spotify Clone Banner](covers.jpg/5.jpg)

---

## ✨ Features

- 🔍 **Search Bar** — "What do you want to play?" live search across all songs & artists
- 💊 **Filter Pills** — Switch between All / Music views
- 🖼️ **14 Circular Cover Cards** — One per artist, with hover play overlay and active animation
- ▶️ **Full Music Player** — Backward · Play/Pause · Forward controls
- ⏱️ **Live Time Display** — Current time and total duration
- 🔊 **Volume Control** — Slider with mute/unmute icon toggle
- ⛶ **Fullscreen Mode** — One-click fullscreen via browser API
- 📄 **About Modal** — Artist list, project stats, credits
- 🖼️ **Dynamic Background** — `bg.jpg` with dark overlay for immersive experience
- 🔖 **Favicon** — Spotify logo shown in browser tab

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/atanu194m/Spotify-Clone.git
   ```
2. Open `index.html` in your browser — **no build step needed!**
